## 部署建议

### 自定义服务器配置

你可以通过以下方式之一进一步自定义：

1. **文件名携带配置**:
   ```bash
   # Windows
   ren remote-assist.exe remote-assist-host=desk.hsid.cc.exe
   
   # 或使用完整配置
   ren remote-assist.exe remote-assist-host=desk.hsid.cc,api=deskapi.hsid.cc,relay=desk.hsid.cc.exe
   ```

2. **配置文件**:
   在程序目录创建配置文件

3. **预配置文件**:
   修改源码中的默认配置

## 注意事项

1. **RS_PUB_KEY**: 需要生成你自己的密钥对，并将公钥填入 config.rs
2. **密码**: 默认密码功能需要额外修改密码生成和验证逻辑
3. **升级检测**: 建议完全禁用升级检测代码，而不是仅仅清空 URL
4. **图标和资源**: 你可能想替换 `res/` 目录下的图标资源
5. **版权声明**: 建议更新版权信息为你的公司信息
