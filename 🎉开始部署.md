## 🚀 立即开始部署（3 步）

### 第 1 步：推送代码到 GitHub（5 分钟）

```bash
cd /workspace

# 初始化 Git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 远程协助 v1.0.0"

# 添加远程仓库（替换成你的！）
git remote add origin https://github.com/你的用户名/你的仓库.git

# 推送
git push -u origin main
```

### 第 2 步：配置 GitHub Actions（2 分钟）

1. 打开 GitHub 仓库页面
2. 点击 **Settings** → **Actions** → **General**
3. 勾选：
   - ✅ Allow all actions and reusable workflows
   - ✅ Read and write permissions
4. 保存

### 第 3 步：等待自动构建（30-45 分钟）

- 推送后会自动触发构建
- Windows: ~20 分钟
- Android: ~25 分钟
